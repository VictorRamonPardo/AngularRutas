Página de parametros con ID {{ id }}

Como podemos observar en la URL, actualmente estamos en la ruta: (localhost:4200/parametros/{{ id }})

En nuestro fichero de rutas (app.routing.ts) hace referencia a: path: 'parametros/:id', component: ParametrosComponent

Con esto nos mostrará lo que tenga el HTML que hemos definido en el fichero: parametros.component.ts

En esta URL le tenemos que pasar un parametro (en este caso el ID) que será el que muestre una cosa u otra dependiendo de dicho parametro.

Podemos pasarle el parametro de dos maneras diferentes:

  • [routerLink]="['parametros/1']
  • [routerLink]="['parametros', 2]
